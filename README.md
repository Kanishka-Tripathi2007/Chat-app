# Chat-app
A responsive chat application built with HTML, CSS, and JavaScript that allows users to send and receive messages through a clean, modern interface. It features chat bubbles, responsive design, and an intuitive user experience.
# 💬 Chat Application

## 🚀 Features

* Modern and responsive UI
* Send and receive messages
* Chat bubble interface
* Auto-scroll to the latest message
* Simple and lightweight design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```
chat-app/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Run `index.html` in your browser.
